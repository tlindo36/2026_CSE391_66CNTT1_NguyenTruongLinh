## 📚 Thông tin môn học: CSE391 -- Nền tảng phát triển Web

### 🎯 Mục tiêu môn học

Môn học cung cấp nền tảng vững chắc về phát triển phần mềm và Web thông qua việc làm quen với các môi trường phát triển chuyên dụng, hệ thống quản lý mã nguồn (Git) cùng cấu trúc cốt lõi của các nền tảng Web.


------------------------------------------------------------------------

### 💻 Kiến thức Web cơ bản trong môn học

Làm quen với kiến trúc Web và các công cụ phát triển,
các công nghệ frontend như:
- HTML
- CSS
- JavaScript
- Frontend framework cơ bản
- Kiến trúc Web: Frontend -- Backend -- Database

------------------------------------------------------------------------

### 🧭 Hoạt động thực hành

Thực hành thông qua:
- Làm việc nhóm trên GitHub
- Tạo repository và quản lý source code
- Xây dựng trang Web cơ bản
- Thiết kế layout, form, validation bằng JS
- Tổ chức cấu trúc project chuẩn

Ví dụ các thao tác Git cơ bản: 
- git add
- git commit
- git push
- git pull
